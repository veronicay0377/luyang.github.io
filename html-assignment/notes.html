Here are 5 of the HTML tags I have learnt:

li
It defines a list item, used within ordered or unordered lists.
img
Embeds an image in the document.
ui
Defines an unordered list, where each item is marked with a bullet point.
ol
Defines an ordered list, where each item is marked with a number or letter.
form
Defines a form, used to collect user input.
